# benchmarks

Benchmark workspace root.

Directory order follows the recommended structure in
`Docs/cpu_gpu_benchmark_tool_plan.md`.

```text
00_system_info
01_cpu_float
02_gpu_float
03_cpu_bandwidth
04_gpu_bandwidth
05_pcie_bandwidth
06_cpu_microbench
07_gpu_microbench
reports
```

Keep runnable harnesses, command logs, raw outputs, and notes in the matching
numbered directory. Keep final summaries in `reports`.
