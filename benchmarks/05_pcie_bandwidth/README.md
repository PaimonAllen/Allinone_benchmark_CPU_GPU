# 05_pcie_bandwidth

Host/device and GPU/GPU transfer bandwidth benchmarks.

Typical contents:

- CUDA `bandwidthTest` runs for Host-to-Device, Device-to-Host, and
  Device-to-Device bandwidth.
- Pinned and pageable memory transfer results.
- `nvbandwidth` runs for multi-GPU systems when available.
- PCIe generation, PCIe width, and topology notes.
