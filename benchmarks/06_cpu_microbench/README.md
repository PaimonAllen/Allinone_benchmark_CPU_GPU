# 06_cpu_microbench

CPU loop, instruction, cache, and latency microbenchmarks.

Typical contents:

- Google Benchmark harnesses for loops, functions, small kernels, and data
  structures.
- LIKWID-bench or `perf stat` outputs for cycles, IPC, cache misses, branch
  misses, and instruction throughput.
- Notes on compiler flags, optimization level, CPU frequency, and pinning.
